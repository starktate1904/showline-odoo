# Preview List PDF

Production-focused module that adds a **Preview PDF** button to list views and renders inline PDF output.

## Features
- License key validation (`XXXX-XXXX-XXXX-XXXX`)
- License status stored in `ir.config_parameter`
- Scheduled re-validation cron
- Backend integration for list PDF preview

## Installation
1. Copy module into custom addons path.
2. Update app list.
3. Install `Preview List PDF`.
4. Go to Settings and set the license key.

## Notes
- Compatible with Odoo 17/18/19.
- Works on Community and Enterprise.
