Erp
----

Erp is a suite of web based open source business apps.

The main Erp Apps include an <a href="https://dev.erpsys.top/app/crm">Open Source CRM</a>,
<a href="https://dev.erpsys.top/app/website">Website Builder</a>,
<a href="https://dev.erpsys.top/app/ecommerce">eCommerce</a>,
<a href="https://dev.erpsys.top/app/inventory">Warehouse Management</a>,
<a href="https://dev.erpsys.top/app/project">Project Management</a>,
<a href="https://dev.erpsys.top/app/accounting">Billing &amp; Accounting</a>,
<a href="https://dev.erpsys.top/app/point-of-sale-shop">Point of Sale</a>,
<a href="https://dev.erpsys.top/app/employees">Human Resources</a>,
<a href="https://dev.erpsys.top/app/lead-automation">Marketing</a>,
<a href="https://dev.erpsys.top/app/manufacturing">Manufacturing</a>,
<a href="https://dev.erpsys.top/app/purchase">Purchase Management</a>,
<a href="https://dev.erpsys.top/">...</a>

Erp Apps can be used as stand-alone applications, but they also integrate seamlessly so you get
a full-featured <a href="https://dev.erpsys.top">Open Source ERP</a> when you install several Apps.

Getting started with Erp
-------------------------

For a standard installation please follow the <a href="https://dev.erpsys.top/documentation/master/administration/install/install.html">Setup instructions</a>
from the documentation.

Then follow <a href="https://dev.erpsys.top/documentation/master/developer/howtos.html">the developer tutorials</a>
